# Archive Directory

This directory contains historical investigation materials and research scripts from the development of the BC MCP Server. These files are retained for reference but are not part of the active codebase.

## Contents

### investigation-docs/
Historical investigation documents, test results, and analysis from the development phase. Most of these are temporary test outputs and debugging logs.

### filter-research/
Research and experiments related to implementing the filter functionality for Business Central list pages.

### analysis-scripts/
Various analysis scripts used during development for investigating BC protocol and WebSocket communication.

### test-scripts/
Legacy test scripts that were used during development. For current tests, see the main test files in the project root.

## Note

- Important documentation has been moved to `/docs`
- This archive is maintained for historical reference only
- None of these files are required for the functioning of the MCP server
- If you need current documentation, check the `/docs` directory

## Cleanup Policy

These files may be deleted in future releases once the project is stable and the historical context is no longer needed.
